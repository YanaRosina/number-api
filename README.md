## Интеграция с http://numbersapi.com, интерфейс для отображения фактов о введенных/ рандомных числах

Стек проекта:

- React, Next js
- TypeScript
- Material UI библиотека

Что бы запустить проект: склонируйте данный репозиторий и выполните npm run dev
